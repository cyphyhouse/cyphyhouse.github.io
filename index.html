CyPhyHouse : a toolchain for distributed robotics

CyPhyHouse aims to provide programming, debugging, and deployment benefits for distributed mobile robotic applications. Develop hardware-agnostic, distributed applications using the high-level, event driven Koord programming language included with CyPhyHouse, without requiring expertise in controller design or distributed network protocols.

Code

Use Koord, our high level programming language to write clear, succinct code.  

Simulate

Simulate Koord applications directly in our simulation environment in Gazebo. 

Deploy

Deploy Koord code on heterogenous hardware platforms. 

Features

Clean Code

Koord abstractions make coding easier by moving low-level hardware functionality and network communication protocols under the hood. 

Learn more »

Modular Design

 CyPhyHouse middleware allows plug and play features such as path planners, controllers, mutual exclusion and communication protocols.

Learn more »

Separation of Concerns

The formal semantics of Koord separates out platform specific control logic from distributed coordination program logic to simplify formal analyses.

Learn more »

High Fidelity 

Applications are deployed as well as simulated, without any modification to code, preserving the behavior of the distributed coordination logic.

Learn more »

Fast Development Cycle

Configuration files, launch protocols and no code recompilation reduce overheads in testing, debugging and deployment 

Learn more »

Customizable 

Koord semantics is highly customizable due to its design, and CyPhyHouse has well defined interfaces to support the customizations. 

Learn more »
