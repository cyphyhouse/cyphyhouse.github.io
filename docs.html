Publications

  • Koord: A Language for Programming and Verifying Distributed Robotics Applications (To appear in OOPSLA'20)

    Ritwika Ghosh, Chiao Hsieh, Sasa Misailovic and Sayan Mitra

    This paper presents Koord—a domain specific language for distributed robotics—which abstracts platform-specific functions for sensing, communication, and low-level control.  (pdf, preprint version)

  • CyPhyHouse: A Programming, Simulation, and Deployment Toolchain for Heterogeneous Distributed Coordination (ICRA'20)

    Ritwika Ghosh, Joao Porto, Chiao Hsieh, Amelia Gosse, Minghao Jiang, Hebron Taylor, Peter Du, Sayan Mitra and Geir Dullerud 

    This paper introduces the CyPhyHouse - a toolchain that aims to provide similar programming, debugging, and deployment benefits for distributed mobile robotic applications.  (pdf, extended version on arXiv)  (video presentation)

  • Language Semantics Driven Design and Formal Analysis for Distributed Cyber-Physical System (ApPLIED'18)

    Ritwika Ghosh, Sasa Misailovic and Sayan Mitra

    This paper presents the executable semantics of Koord in the K framework, including tools for formal analyses. (pdf)

  • StarL: Towards a Unified Framework for Programming, Simulating and Verifying Distributed Robotic Systems (LCTES '15)

    Yixiao Lin and Sayan Mitra

    The StarL framework for programming, simulating, and verifying distributed systems that interacts with physical processes is the precursor to CyPhyHouse, and a Java alternative to our Python middleware. (pdf)
